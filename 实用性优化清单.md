## 建议执行方案

总工时约 **1 小时**：

```
1. 后端密码长度校验             ← 5min  (安全底线)
2. 后端 health endpoint         ← 10min (运营需要)
3. 日历「今天」按钮             ← 15min (高频使用)
4. SVG favicon                  ← 10min (感知质量)
5. StatsCards loading skeleton  ← 10min (避免用户疑惑)
6. 启动时输出数据库路径          ← 5min  (调试友好)
```

其余工程化项目（ESLint、Docker、PG 迁移、API 拆分）**现阶段都不值得做**。
